# 充值方式

## 苹果手机

### 使用礼品卡充值

1\. 打开苹果官网礼品卡页面：<https://www.apple.com/shop/gift-cards>。
2\. 选择礼品卡发送方式为 **Email**。
3\. 选择任意礼品卡封面。
4\. 选择或填写充值金额。
5\. 填写收件人信息：
   - 收件人姓名：可自行填写
   - 收件人邮箱：可填写美区 Apple ID 对应邮箱
6\. 填写发送人姓名和邮箱。
7\. 点击 **Add to Bag**，将礼品卡加入购物袋。
8\. 进入购物袋后，点击结账。
9\. 选择 **Guest Checkout**，使用游客身份结账。
10\. 支付方式选择信用卡。建议使用支持 Mastercard 或 Visa 的外币信用卡。
11\. 填写信用卡信息：
```java
Card Number：信用卡号
Expiration Date：有效期
CVV：安全码
```
12\. 填写账单地址，例如：
```java
Country / Region：United States
First Name：ax
Last Name：haixxxxx
Street Address：853 Northwest Kansas Avenue
ZIP Code：97702
City：Bend
State：OR
Phone Number：458 7022953
```
13\. 确认订单信息无误后完成支付。
14\. 支付成功后，查收邮件中的礼品卡兑换码。
15\. 在 iPhone 上登录对应的美区 Apple ID，进入账户兑换礼品卡金额。

兑换完成后，即可使用 Apple ID 余额订阅或续费 ChatGPT。
