

下面给你一份**Mac 终端（Terminal / iTerm2）常用快捷键大全**，按“编辑 / 光标 / 历史 / 控制”分类整理，适合开发者直接用👇

---

# 🧠 一、光标移动（最常用）

|快捷键|作用|记忆方式|
|---|---|---|
|`Ctrl + A`|移动到行首|A = Ahead|
|`Ctrl + E`|移动到行尾|E = End|
|`Option + ←`|向左跳一个单词|按词移动|
|`Option + →`|向右跳一个单词|按词移动|

---

# ✏️ 二、删除 / 编辑（高频）

|快捷键|作用|记忆方式|
|---|---|---|
|`Ctrl + U`|删除光标前整行|U = Up / Undo 前面|
|`Ctrl + K`|删除光标后整行|K = Kill 后面|
|`Ctrl + W`|删除一个单词|W = Word|
|`Option + Delete`|删除前一个单词|按词删|
|`Delete`|删除单个字符|基础|

---

# 🔍 三、历史命令（核心提效）

| 快捷键        | 作用             |
| ---------- | -------------- |
| `Ctrl + R` | 🔥 搜索历史命令（最重要） |
| ↑ / ↓      | 切换上一条/下一条命令    |

👉 Ctrl + R 是开发者必会技能

---

# ⚡ 四、命令控制

|快捷键|作用|
|---|---|
|`Ctrl + C`|中断当前命令|
|`Ctrl + Z`|挂起当前进程|
|`Ctrl + D`|退出 shell / EOF|
|`Ctrl + L`|清屏（等价 clear）|

---

# 🧹 五、快速操作组合

### 🚀 清空当前命令（经典）

```text
Ctrl + U
```

---

### 🚀 彻底重写命令

```text
Ctrl + A → Ctrl + K
```

---

### 🚀 快速修复命令

```text
Ctrl + A → 修改 → 执行
```

---

# 🧠 六、Tab 自动补全（必须用）

|操作|作用|
|---|---|
|`Tab`|自动补全命令 / 文件|
|`Tab Tab`|展示所有候选|

---

# ⚡ 七、进阶（开发者很有用）

|快捷键|作用|
|---|---|
|`Ctrl + T`|交换字符（少用但有趣）|
|`Ctrl + _`|撤销操作（部分 shell 支持）|

---

# 🧩 八、实战组合（最重要）

### 🔥 查历史 + 执行

```text
Ctrl + R → 输入 docker → 回车
```

---

### 🔥 修改错误命令

```text
Ctrl + A → Option + Delete → 修改
```

---

### 🔥 终端清理

```text
Ctrl + L
```

---

# 🧠 九、核心记忆口诀（建议背这个）

> **A头E尾，U前K后，W单词，R历史，C中断，L清屏**

---

# 🚀 十、总结（开发者视角）

终端效率核心就三件事：

- 🔍 Ctrl + R（找历史）
    
- ✏️ Ctrl + U / K（快速删）
    
- ⚡ Ctrl + A / E（快速定位）
